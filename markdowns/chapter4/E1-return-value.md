# Return Value

```C++ runnable
# include <iostream>
using namespace std;

// function return_two returns an int variable that equals 2
int return_two() {
  return 2;
}

int main() {
  cout << 1 << endl;
  cout << return_two() << endl;  // print returned value
  cout << 3 << endl;
  
  return 0;
}
```
